中環華賢坊東

不完美之美

Title: #self

你的道德核心： 追求 “卓越 与 真实” - 价值增量： 道德的行为， 应当是 让双方都变得更好 - 智识诚实： 有着近乎苛刻的客观要求。 不喜欢模糊的掩盖，更倾向于直面问题

 - 对他人的底线：  拒绝 “不对称操纵” 与 “情绪绑架”
 - 对自己的底线：  拒绝 “平庸” 与 “虚度”

 - 需要警惕“灰色地带”
    +  从“自我实现” 向 “社会连结”过度的关键期
    +  自我保护与防御：  你习惯于用“逻辑体系” 来保护自己， 
        当感到不安全时，可能退汇到你的“逻辑壳”里。 
        如果不小心，会变成拒绝真诚的一道墙

    +  不要让“逻辑的底线” 变成 情感的冷漠“
         正直要求你对事客观，但真诚要求你对人有温度

正直 consistency： 是否“可预测” - 保持底线不变的情况下， 用极其灵活和高效的方式处理

  - 一致性： 知性合一； 你的内在准则与外在行为保持同步

      +  言行一致： 说到做到

      +  标准一致： 不会因为对象不同而改变原则
           -  区分 准则 和  处事方式

      + 情绪一致：

 - 底线： 
      + 可以极其精明地追求利益， 但手段永远不触碰你设定的底线
