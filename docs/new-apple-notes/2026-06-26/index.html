反省過度， 不夠自由

自信 + 反省 - 小事可以自信到底， 但生死關頭就要跳出來， 跳出自身环境来客观分析，究竟自己堅信的是否真正正確？

-  在做任何重大決定前， 必須給自己訂下數個基本假設
     +   大事如選擇 職業， 娶妻， 作出重大投資， 移民等必須考慮清楚

-  加入基本假設不變，就鍥而不捨地繼續；
    只要這些假設一改變，就從速反省，作出對策

自信 - 自信來自安全感

- 在現實世界中， 什麼是對，什麼是錯，絕大多數情形下， 我們不會知道

- 一個人如果欠缺自信，就會拿不定主意，不能決定任何事，也就沒法採取任何行動。
其實只要有行動，即使錯了，你也知道錯在哪裡，又或者可能未必全錯，有機會部分是正確的。
 不過，如果舉棋不定，那就沒有機會做對，也不知道出了什麼錯。
       + 堅決地有所行動才是成功的要素。

-  必須明白自信心的來源在哪裡？
       +  坚定不移的原则 <- 理性分析而来
           -  做重大决定前，**必须给自己几个基本假设**  
           - 只要这些假设一改变，就从速反省，作出对策

       + 自己比别人努力和专注

你的道德核心： 追求 “卓越 与 真实” - 价值增量： 道德的行为， 应当是 让双方都变得更好 - 智识诚实： 有着近乎苛刻的客观要求。 不喜欢模糊的掩盖，更倾向于直面问题

 - 对他人的底线：  拒绝 “不对称操纵” 与 “情绪绑架”
 - 对自己的底线：  拒绝 “平庸” 与 “虚度”

 - 需要警惕“灰色地带”
    +  从“自我实现” 向 “社会连结”过度的关键期
    +  自我保护与防御：  你习惯于用“逻辑体系” 来保护自己， 
        当感到不安全时，可能退汇到你的“逻辑壳”里。 
        如果不小心，会变成拒绝真诚的一道墙

    +  不要让“逻辑的底线” 变成 “情感的冷漠“
         正直要求你对事客观，但真诚要求你对人有温度

正直 consistency： 是否“可预测” - 保持底线不变的情况下， 用极其灵活和高效的方式处理

  - 一致性： 知性合一； 你的内在准则与外在行为保持同步

      +  言行一致： 说到做到

      +  标准一致： 不会因为对象不同而改变原则
           -  区分 准则 和  处事方式

      + 情绪一致：

 - 底线： 
      + 可以极其精明地追求利益， 但手段永远不触碰你设定的底线

當下即是完整 - 完整意味著 自在， 自足， 幸福快樂， 都是內在感受，而不是外在的人事物； 只要你有能力為自己創造自在，自足，快樂，何必要等？ 又何必為自己活別人貼上“不完整”的標籤？

- 從此刻開始， 人生就是你自己的，不論任何遭遇， 你都是完整的， 不完整的是你的念頭

積極主動找朋友，熱情地接待他們，並考慮他人感受 - 志趣相投， 大家談的內容，交流心得要一致

朋友 - 忠直： 交朋友要真心真意， 不然交了虛偽的人沒意思 - 要互相諒解，不能記仇： 如果不能諒解你的人， 也是不能做朋友的 - 結交有學識的人， 能夠在學業或事業上助你一把

貧視其所不取 - 指一個人貧窮時，要看他不取哪些東西。

富視其所與 - 看一個人有錢後，把錢用在甚麼地方？

達視其所舉 - 要看他舉薦和重用哪些人

不能胡亂信人 - 要信任一個人，一定要那人久經考驗，確認可信才行。 也正因為考核嚴苛，每個人的一生可能來來去去只有極少數可信任的人 - 不要借錢給朋友

不能結交的人 - 阿諛奉承的 - 跟以前所有的朋友都鬧翻收場 - 經常懷有否面思想的 - 老在背後說人壞話

生意夥伴 - 能夠補充自己不足，有信用及互相體諒的人

不想負責，害怕承擔責任或背鍋 - 盲點： 你認為如果結果不好，大家埋怨你，這個“埋怨” 是你必須承擔的後果 + 區分 “責任” 與 “情緒” + 你對項目成敗負責，但你不需要對他人因結果而產生的負面情緒負責； 別人的埋怨，是他們處理挫折的方式，不是你做錯事的證據 - 風險： 為了躲避“背鍋”的風險， 你很容易陷入“不做不錯”的陷阱；這會讓你喪失主動權， 將自己的未來交給了別人， 而別人往往並不關心你的核心目標

  • 聰明地承擔責任 + 從“決策者” 轉型為 “風險評估者” <- “共识会议” - 將決策變成一場 “共識會議” - 在做重大決策時， 主動陳述你的分析過程與假設， 並問他人： “基於目前的資訊，我傾向採取方案A, 大家有什麼風險預判嗎？”
    - 當你把決策過程公開化， 責任就從“一個人決策” 變成“集體的風險評估”。 如果結果不好，大家討論的是“當時資訊的局限性”， 而不是“你這個人的錯”

              - note： 分享利益，独立决定

                       +  所有的决定都是微观的，是在具体的环境里面对具体问题做出的

          - note: 负责人 先征求上下左右相关人意见，拿出一个方案
                       +  主要看 有无遗漏，有无质疑的声音
                       +   如有质疑，评估一下有无道理； 然后负责人自己就把这件事决定了

                       + 做决策并不想一次性解决所有问题, 而是**追求在原有基础上取得进步**, 有瑕疵也没关系
                              - 独立决断背后是每一个人本身的责任

   +   建立“防火墙” - 防背锅术
          - 害怕别人推卸责任， 通常是因为缺乏明确的交付机制

          - 做法： 在 任何合作中， 利用文字明确权责， 只需要清晰。
            在事情开始前问一句： “这部分最终谁把关？”

    +  处理“被埋怨” 心理成本
          - 这可能源于一种深层信念：  我需要得到所有人认可，才算安全
          -  下次被埋怨： 试着想 “他现在感到焦虑，所以透过指责来释放压力”；
            将他的情绪还给他，将事情的逻辑留在自己手里。

  • 主动承担 一件原本可以推给别人的琐事 + 重点不是结果多么完美， 而是你在这件事中，行使了你的决定权，并承担了过程中的所有变数

            -   完成后， 你感受到了什么？  (紧张，兴奋， 还是掌控感？)
        -   即使结果不完美，天塌下来了吗？

   + 自信不是“我不犯错”， 而是“即便我犯错，我也完全有能力处理后续，并且不让这种失败定义我的人格”

    • Power thinking - 自我信念

      • 我愿意讲出来 我最后会得到足够祝福同支持 我觉得信念变成是事实

      • 无论世界如何，我都能做到能做的事

      • 在不受控因素存在下， 仍然把受控的事做好

当我们观察，了解这个世界的时候，一定要靠更多的事实，经验做基础， <- 归纳，而不是推理 而不是急于把它套在一个很大的理论框架，概念上

很多大的成败都不是可以简单归结于一般性的原因，很多具体的原因 很多细小的事情决定成败，后来的成功失败又重新塑造了人们对历史对社会对人生的看法

所有的决定都是微观的，是在具体的环境里面对具体问题做出的 - 提高整体认知，选择做当下最适合自己，最值得做的事
- 要重视，挑选和管理的，就是周围的 “五六个人”
- 最看重的， 是 是否我愿意与之一起工作的那一类人

    • 有了对自己更好的认识， 自信而不自负

通过做事来 “准备”的， 而不是通过“准备”来做事的 - 面对机会，没准备好不是问题， 要勇敢把握， 多做几次就好了 <- accountability
