我想要什么样的生活

2015-08-16

在大公司，小公司(startup)，学校的RA 都待了一遍，什么是觉得还不错的工作呢？

  • 做的事情有前景，有意义，有点挑战力
  • 老板，有相似的价值观，有能力并且愿意思考下属的发展空间
  • 尊重工程师的view
  • 稍微有些掌控力和自由度
  • 待遇還行

想要什么样的生活呢？

  • 稍微自由（时间，掌控力，财务）
  • 亲密的家人和朋友
  • 自己喜欢做的事
  • 能为社会做点力所能及的贡献
