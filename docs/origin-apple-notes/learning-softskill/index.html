learning softskill

#learning #softskill 什么时候觉得孤单？ <- 给别人一个好/正面的回应

不断的努力，给出好的回应 关系是每一天的小事的经历， 你来我往

连结， 交往， 而不是 沟通

自己的主动性

关系是需要经营的，是个决定

言语或行动邀请 或 回应
因为爱对方， 所以有这个选择 - 正面的回应， 所以忍受

**从小事开始， 接纳她**

“我都觉得我们开始的时候，我都好开心； 我都想 我们再一起努力，再试”

“你好吗”，“你怎么啦”， 或者行动 “一杯水” "今天我有些累，我今天不能听你的，改天我再听你的"

"我觉得我们现在都有情绪， 我们先暂停吧， 改天再讨论"

“可以告诉我到底发生了什么吗” “我有些生气，因为你 ...”

“我们可以一起努力吗”

学 如何 适当回应 别人
只要是个正面的回应就ok 不要有 负面的不好的回应

享受你来我往的过程 ， 比内容更重要

"你觉得我应该送你回家 还是 ..."

对别人有兴趣

有困难，依然在乎你

爱自己， 把最好的自己送给爱人 互相努力，培训而成
