math insight

math #insight

  1. 数学不是万能的 - 有意义的问题
  2. 个体层面 - 懂得数学的思维方式 - 直觉，辨析（进一步寻找证据，进一步肯定或否定），逻辑 - 从“发散” 到 “不完全归纳” 再到 多角度“试错（证伪）” 到 “肯定”的过程
  3. 运用数学 - 抽象和量化现实 - 从现实-> 数学模型 -> 在数学世界里寻找 -> 回到现实
  4. 数学好比催化剂， 可以改变化学反应速率而不改变化学平衡； 同时催化剂本身也可能发生改变
  5. 数学教育是教人学会思考与解决问题
