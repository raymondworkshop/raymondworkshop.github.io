softskills

#softskills #self

自己很关注对方是不是愿意付出， 而不是情绪，逻辑

说明自己是一个 重结果，讲究对等价值的人 - 在技术协作或商业合作中是极佳品质， 但在处理家人和朋友关系上，容易变成双刃剑

从“交易思维” 转化为 “连接思维” - 为什么会关注“付出” - 潜意识其实是在寻找一种 “安全感” 和 “确定感” - 公平感的渴求： 你可能认为，一段健康的关系应该是有来有往的平衡。 如果对方没有付出，你就感到自己在被“剥削”

     -  评价体系的迁移： 如果将工作/投资 直接搬到人际关系中，你其实是在把“爱”当作一种投资项目

     - 防御机制：  关注对方的付出，其实是在给自己留后路 - 避免自己受伤
  • 关注“付出” 带来的隐形代价 - 把‘人“变成”会计“： 你么之间变成了一种基于积分的交易。 这会让对方感到一种莫名的”被审查感“
    - 遮蔽了”脆弱感“： 人的付出是多样的。 有些人是行为，有些人是精神

- 关系的”零和博弈“： 一旦你开始计较，你的大脑会忽略对方的细微示好，转而放大对方的忽视

加一个过滤器 - 识别“付出”的多种维度 - 有些人付出的是时间，有些人付出的是情绪价值， 有些人付出的是资源

- 从“审计” <b><u>转向 “赋能”</u></b>
    - 问”<b><u>我希望得到什么样的支持？</u></b> 我有没有主动邀请他进入我的世界？“ ， 不要为“他为什么不付出？”

区分 “投资” 与 “经营” - 投资逻辑： 投入资源 -> 观察回报 -> 决定是否增持

- 经营逻辑：   建立愿景 ->  <b><u>共同投入 ->  创造超越个体价值的连结</u></b>
    - 在重要的人际关系中，  把关注点从 ”她为我做了什么？“ 转移到 ”<b><u>我们在一起时， 这种体验质量如何</u></b>？“
