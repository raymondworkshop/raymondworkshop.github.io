softskills work

#softskills #work

这根本不是人际关系的问题，更不是领导能力的问题，而就是团队成员不负责任，没有职业素养，而你是没错的。你在人际交往，领袖能力上，并没有错，更没有什么不足。

分清楚自己哪里做的不对，哪里做的很对，避免不必要的反思，非常重要。

2，最后一集，组织一场电子竞技大会，会场很大，客流量很多，自己带着三个猪队友。

首先是对队友进行分析，分清哪些是平庸但是踏实肯干的，哪些是刺头不听调遣的。

然后是尽可能团结踏实肯干的队员， 接着，在整体创意和布置安排上，自己一个人拿主意，快速的确定方向，做好决策，让平庸的队友只要执行即可，

最困难的活，都是自己干。

再然后，和队友沟通时，不是发号施令，而是细声软语，耐心劝诫，感激鼓励，

对于队友的任何成绩，都是鼓励表扬。

最后完美完成项目，泪洒现场，和队友们也结下了深厚友谊。

这说明在项目创意，调度协调，统筹安排上，她有着不可思议的能力。

以上两个案例说明啥？

在人际关系中，处处受欢迎，几乎是不可能的。

搞好人际关系，只要三分力就够了，对他人尊重，平等足以，用不着热心，主动关怀。

最重要的，还是自己有没有过人的特长，资源，能力，技能，经验等等。

后者，在商业社会里，来的更加重要，而且有了后者，只要你的性格不是那么过分，自然会有一堆同伴来追随。
