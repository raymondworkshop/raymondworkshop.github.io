project-based

#project-based

项目式学习是有一个目标 （可以是清楚的，也可以只有大概方向） - 学习的目的是达到目标，而途径可以不同

  • 关键点 - 设定目标， 制定计划， 获取信息， 团队协作， 测量数据，分析总结， 演示结果

  • 类似爵士乐， 允许演奏者随意发挥，同时要达到预期的效果

  • 完善的计划， 足够的灵活， 一个可以开展团队协作的环境

#self 1. 工作， 尽可能快，尽可能高效      核心任務 - 至少用80%的时间 解決最重要，最困难的部分 - 不追求完美，只需要满足最小可行产品原则 (mini viable product)

2.  休闲， 以享乐主义的视角 - 尽可能享受当下的闲暇     体力活动 + 发展一项爱好，深度沉浸

3.  识别并规划更长收益的事情     - 锻炼身体     - 学习 “繁殖力”的知识     - 获得巅峰人生体验     - 从事写作等创造性活动     - 提升个人审美     - 反思与总结
