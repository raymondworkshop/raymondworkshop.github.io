个人复杂技能

#个人复杂技能

  1. 思考我写书的优势和本书的独到之处在哪里，我给读者提供的价值在哪里 - 想清楚之后，再围绕着几点来写

  2. 再思考能否做成 - 索引 涵盖希望讲的最重要的几个方面 - “渐进的过程”， 只要知道需要填补哪些空白就行

  3. 粗略地填写我有想法但没写过的章节 - 最小产品化

  4. 大量细化和整理

团队负责技能

  1. 场景训练
  2. 实战分析
  3. "人才池" - meetup

不仅仅是单纯的技能训练， 变成复杂的场景培训，战术培训，能够做到总体上的提升

解决复杂问题的能力

  1. 分析问题 - 预测问题的复杂程度和工作量
  2. 分解问题
  3. 看出关键点，各个击破

#self

  • 先做好一件事
  • 再学会“知道你的角色和团队协作”
  • 学会“如何与不同的人沟通” - 学习如何说服别人，学会应付碰壁，走不通的情景
  • 领导力
