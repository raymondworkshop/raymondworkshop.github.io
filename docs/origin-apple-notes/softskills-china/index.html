softskills china

#softskills #china

中国的文化意识 - “争表现”心理 1. 很少有人从小就有独立的思想，把自己和来自父母老师的奖赏分离，为自己考虑 2. 自信，有思想，有尊严 - 有自己的尊严和规则 - 受人尊重的成年人

#softskills #important 待人接物 -> 彼此尊重、亲疏随缘 1. 待人 - 对待亲朋好友 - > 需用感性 - 感性支持 2. 接物 - 对待社会人陌生人给的事情，任务，-> 需用理性 - 理性判断 -> 实现个人目标的工具

社会人事 -> 对理性控制性格 -> 将外部目标转化为自己控制的环节，形成自我驱动习惯 亲朋好友 -> 对感性表达性格 -> 将外部目标转化为与人同行的氛围

不同的场景，遇到不同性格的人，采用不同的打开方式 人之美，非大全而独美，而在协同与众之大美。如能做到这样，自己舒服愉快，别人也愿意接近同行。

不同的任务环境里不同的待人待己方式 短期情感 - 比如亲友聚会，同学聚会，公司团建 长期情感 - 亲密关系，理想导向的团体关系 工作沟通 - 公司见面打招呼，各个部门同事接触 工作项目 - 一年目标，三年规划
