social-skills

#social-skills - 缺少正常人际交往边界意识 -> 人与人相处，各自有个边界，越过了那个边界，对方就会觉得不舒服 - 比较自恋，控制性强，要左右别人如何想，如何做

#learning - 不管做什么题目，先问自己： 要解决什么问题？ 如果别人已经解决过了，要搞清楚，学会了，再去创新。 如果要创新，提出新的解决方案，要问自己，你的解决方案是不是比别人的更高明？

  • “把别人已经证明有用的东西学会了”

  • 缺乏生活世界的常识和对世事的观察能力

  • 找出核心概念，推论的大前提小前提，看每个章节的结论成立不

  • 量才适性，自己有多少本事，尽量做自己该做的事； 有社会责任感，但事责任感不能无限膨胀

  • 要谦虚的读书

#obversation 国家认同和价值观认同比民族和种族认同更能影响敌友

#insight 现代文明就是从传统文明那些好的东西中演进出来，是一种再创造，或者说新生。人类历史上，不只是中国经历过传统文明 ---- 那不是一种独特的经历
