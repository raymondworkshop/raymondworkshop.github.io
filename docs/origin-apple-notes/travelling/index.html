travelling

#travelling

尽量不回应

深圳行： 大芬油画村 + 大梅沙海濱棧道（大梅沙->盐田海鲜街, 8公里， 2小时）

準備 - 出入境健康申报 （wechat） - 回乡证/港澳通行证 - 深圳通 （類似八達通， @羅湖地鐵站） or 人民币

大致行程 draft： 1. ~10:00 上水地鐵站集合； 上水站=>羅湖站 （10分钟）

  1. ~12:00 大芬油画村； 地铁1号线 → 地铁3号线 （大芬站（A1口））  （40分钟） 1 小时 休閒

    ~13:30 Lunch - Any suggestions - 金丰港式茶餐厅 - 日式料理 - 海南酸粉

      - 木屋烧烤（大芬店）?

  2. ~15:00 到 盐田海鲜街； 大芬油画村 => 大梅沙海滨公园

    • 地铁3号线 → M380路 （1小时30分钟） + 地铁3号线 （大芬站  - 塘坑站 （C口) ） + M380路 （塘坑地铁站  -> 梅沙街道办站 ）

    • 地铁3号线 → 地铁5号线 → 地铁8号线 → M196路 + M196路  （盐田港西地铁站  -> 中兴发展站)

下午 - hiking -2hour 大梅沙海滨公园 -> 盐田海鲜食街

~ 18:00 Dinner @ 盐田海鲜街 - Any suggestions - 东腾海鲜酒楼 ?

  1. 返回，自由时间 ~18:00 盐田海鲜街 -> 罗湖站 M205路 （1小时24分）
