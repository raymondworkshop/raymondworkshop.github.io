做风险决定的框架
- 分析利弊， 尤其是自己比较关心的地方
- 定义成功的标准， 最好量化
- 分析成功的概率
- 如果结果不尽人意， 还能获得什么？
+ 比如有了这些收获，失败也是一种成功
+ 比如做这件事成本不算大
+ 失败情况下收获依旧很大 - 继续走下去， 所要付出的情绪和经济成本有多少？
