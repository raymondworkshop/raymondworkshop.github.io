真正的自主性並非排斥責任，而是選擇在哪裡，以何种能力去承担责任，
而这种选择需要与群体追求的共同意义相匹配
    - 真正的连接 需要 “共同追求意义”

熟人社会里的独立个体
    - 学会使用“平行动作” 拒绝上下级标签
           +  用“讨论” 代替 “服从”， 把对方的 “命令” 降格为 “建议”
           + “这个想法挺有意思。 不过我觉得我们也可以试试另一种方式 …”

           +  当别人试图把责任推给你时 （把你当上级）
           +   及时把球踢回去， 明确 “课题分离”， 不主动承揽别人的责任
           +  “这次活动大家都出点力，不如我们把任务分一下， 你负责A, 我负责B, 怎么样？”

工作中的 “上下级”
     - 上司是你的“大客户”， 他不是你的“主人”
           + 把上司 “命令” 看作 “需求说明书” 
                -  用专业视角去 “管理预期” ，协助对方做决策
                - “如果按照这个方案做，可能会带来A和B的风险。 我建议另外两个优化路径，你看哪个更符合预期？”

     - 下级 是  “项目外包商”， 你不是他们的“保姆”  
            + 用流程和机制去约束， 不要用情感去管理
                - 建立“带着方案来敲门”的文化； 永远不要直接给答案。 你一给答案，责任就变成你的了
                -  “这个问题确实棘手。 不过我想先听听，你目前想到的两个解决方案是什么？ 它们的优缺点你评估过吗？

            +  明确责任和权力边界： 在派发任务时， 把过程责任和最终责任分清楚
                - 这个项目的过程推进和执行决策由你全权负责，如果搞砸了，你需要承担相应的绩效后果。 
                   我作为上级，只负责在必要时提供资源支持

你即要有主见， 又要接受反馈，把握好平衡是关键

面对一个问题，你需要想好怎么做，然后主动向老板提议这么做，再问他有什么想法。
    -  如果这个问题没有完美解法， 你需要想好多种做法，对比它们的优缺点，
然后陈列给老板看，指出你推荐的做法，但让他做最终选择。 这是平衡的做法

    - 想好了怎么做就立即动手做，不问老板，这是有风险的。
    你需要提前想好， 哪些事情老板只看结果不看过程，
    哪些事情老板不同意你的trade-off 你直接做了就会受到惩罚

    - 想都不想就问老板要解法则是懒惰，除非这是老板知道远超你能力范围的工作。
       如果你和老板对于你“能力范围”的期望不一致，你同样会死

* 如何了解对方的心态
    - 自己的心态： “我分享内容是因为这事很有趣，你回不回我不受影响”

    - 回应的速度
        +  秒回， 几小时内回复，  隔天或长期冷处理

    - 回复的厚度  
        +  字数与你相当， 回复字数多于你，  惜字如金

    -  字里行间的“温度”
        +   互动性强： 反问， 自嘲， 吐槽

        +   防御性强： 客套/礼貌

    -  进阶： 如何了解对方的心态？
        +   放置钩子：   在回复中主动分享一个关于自己的，“稍微带点个人特色”的信息

        + 观察对方的回应：
              - 高兴趣：  追问细节，或分享她遇到的类似难题  （心态： 对你好奇，想了解你的精神世界）

              - 礼貌忽略

              - 情绪共鸣： 分享了自己的挫折或感悟 （心态： 视你为可以分享情绪的伙伴）

连接 是靠 “自我建设 + 适当暴露 + 提供支撑” 产生的
     -  同频 -  认知同步 - 思考逻辑，价值观，洞察力
          +  谈“底层逻辑”， 当对方发现你能站在和她一样的维度思考问题时， 共振自然发生
          +   延伸话题：  你抛出一个观点， 对方是能提出一个新视角或反问， 把对方推向更深处
          +  沉默不尴尬：  “舒适的沉默”

     -  通过“个人分享” 加速连接 Strategic Vulnerability -  不要做一个完美的人 
         +   分享一个你的小挫折。  这种真实的挫败感
         +  自我暴露：   对方开始讲自己工作中的失误，和家人的分歧，对未来的不确定性
         +  信任测试：   对方分享了一些未公开的计划或琐事。 这是对方把你从“普通朋友” 列入“知己”信号

     -   “情绪价值”  -  肯见与肯定 validation  
         +   观察她没说出口的压力 

         +   这种被“看见”的感觉， 是男人能给予女性最强的能量补给
               - “我知道...,  辛苦了”

          +  回馈关心：  当你遇到困难时， 她不仅提供解决方案，还会主动询问你的感受 （这是顶级同理心的表现）

    - “连接法则”
         +   主动权平衡  
                -   如果你发现对方总是冷处理你的“高价值话题”； 
                不要纠结，退回一步， 把重心转回到你自己的项目建设上，保持吸引力

         +   话题阶梯
                -  安全区：  天气， 爱好
                -  建立信任：  共同目标， 趣事
                -  深层连接：  挫折，价值观， 未来

        +  情绪留白：   不要把话聊死。 留给对方“主动找你”的机会
